# Iron Gym OS V2 Roadmap

## Phase 0 - Deployment foundation
- GitHub repository created.
- Vercel deployment connected.
- Supabase project connected through environment variables.
- Migration 001-002 prepared in Supabase.

## Phase 1 - Auth and OWNER bootstrap
- Add email/password login.
- Add first-account OWNER bootstrap through `bootstrap_current_user_as_owner` RPC.
- Confirm the authenticated owner can read their app profile.

## Phase 2 - Core Admin dashboard
- Branch selector.
- Staff and role management.
- Package product management.
- System health/status page.

## Phase 3 - Customer OS
- Customer profile CRUD.
- Health profile.
- Body composition history.
- Body measurement history.
- Trainer assignment.

## Phase 4 - Sales OS
- Multi-item invoice creation.
- Membership + PT + PL + KF in one bill.
- Payment history.
- Customer package generation from invoice items.

## Phase 5 - Training OS
- Training session booking/logging.
- Package session deduction.
- Google Calendar automation through Apps Script.

## Phase 6 - Automation + Game Mode
- LINE reports.
- PDF/export.
- EXP, levels, ranks and gamified dashboards.
