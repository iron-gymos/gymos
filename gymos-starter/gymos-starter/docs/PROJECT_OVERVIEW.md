# Iron Gym OS V2 Project Overview

## Direction

Iron Gym OS V2 is a new clean rebuild. Supabase is the main database and source of truth. Google Apps Script will be used later only as an automation layer for Google Calendar, LINE reports, PDF and export jobs.

## Roles

- OWNER
- SUPER_ADMIN
- ADMIN
- MANAGER
- ASSISTANT_MANAGER
- SALE
- TRAINER

## Core V2 Scope

- Multi-branch gym management
- Detailed customer records
- Health profile, injuries and training cautions
- Body composition and body measurement history
- Membership and training package separation
- Multi-item invoice flow
- Training sessions and Google Calendar event reference
- Automation event outbox for Apps Script integrations
